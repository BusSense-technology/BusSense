# 📖 Sobre o Projeto

A **BusSense** é uma solução de tecnologia assistiva criada para aumentar a autonomia de pessoas com deficiência visual no transporte público.

Nosso sistema integra dados em tempo real da **API Olho Vivo (SPTrans)** com um dispositivo wearable inteligente, fornecendo sinais táteis e sonoros que auxiliam o usuário durante seus deslocamentos urbanos.

O projeto nasceu com o propósito de transformar dados urbanos em acessibilidade, inclusão e segurança.

---

# 🚀 Nossa Missão

Promover uma mobilidade urbana mais inclusiva através da tecnologia, oferecendo mais independência, acessibilidade e segurança para pessoas com deficiência visual.

---

# 💡 Como Funciona

## 🔹 Integração em Tempo Real

Recebemos informações da API da SPTrans para identificar ônibus, linhas e rotas em tempo real.

## 🔹 Processamento Inteligente

Nosso backend interpreta os dados e converte as informações em comandos acessíveis.

## 🔹 Tecnologia Wearable

O usuário recebe vibrações e sinais sonoros através de um dispositivo sem tela, permitindo orientação intuitiva.
